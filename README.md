# BBC-Style News Website

A modern, responsive news website built with HTML, CSS, and JavaScript, featuring a clean design inspired by BBC's layout and functionality.

## 🌟 Features

### Core Features
- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **News Integration**: Real-time news fetching from external APIs
- **User Authentication**: Registration and login system with local storage
- **Navigation**: Sticky header with mobile-friendly navigation menu
- **Pagination**: News pagination for better content organization
- **Cross-page Navigation**: Seamless navigation between different sections

### Pages
- **Home Page** (`index.html`): Main landing page with featured news
- **News Page** (`news.html`): Dedicated news section with pagination
- **Sports Page** (`sports.html`): Sports-specific news content
- **Login/Register Page** (`loginpage.html`): User authentication system

### Technical Features
- **Sticky Header**: Header and navigation bar that stays at the top when scrolling
- **Mobile Menu**: Hamburger menu for mobile devices
- **Form Validation**: Client-side validation for user inputs
- **Local Storage**: Persistent user sessions and data storage
- **API Integration**: News API integration for dynamic content
- **CSS Grid & Flexbox**: Modern layout techniques for responsive design

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML, CSS, and JavaScript
- Text editor (VS Code, Sublime Text, etc.)

### Installation
1. Clone or download the project files
2. Open the project folder in your text editor
3. Open `index.html` in your web browser to view the website

### Running the Project
- **Local Development**: Simply open `index.html` in your browser
- **Live Server**: Use VS Code's Live Server extension for development
- **Local Server**: Use any local server (Python, Node.js, etc.)
